# RaptorsTicketApp Project


## Description
It is the Raptors Tickets app that a user can use to purchase tickets depending on the number of tickets and their respective prices.

## Functionalities

* MVC pattern.
* Prepare Function.
* Delegates and DataSource.
* TableView.
* Pickerview Controller.
* Push segue and Module Segue.
